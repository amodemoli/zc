## The Zima Commands Application
zima console commands app, this application maded for use easy linux and windows console.  <br>
this application add new simple commands for user daily usage. <br>
zima app coded by *Demolition* and used *golang* in project. <br>
you dont need goland compiler for install application. (compiler only needs in build app) <br>

[»] **Install In Linux**:
1) you need to download `/build` directory first.
2) open terminal in file and run this commands:
```bash
bash ./install.sh 
```
3) application installed on your system
4) you need *sudo for install application <br>

[»] **Install In Windows**

1) you need to download `/build` directory first.
2) open `install.bat` file with adminstrator perms.
3) application installed on your system.
4) if you open 1.0 version installer in powershel you cant see colors. (this bug fixes in next version)
