## The Zima Commands Application
zima console commands app, this application maded for use easy linux and windows console.  <br>
this application add new simple commands for user daily usage. <br>
zima app coded by *Demolition* and used *golang* in project. <br>
you dont need goland compiler for install application. (compiler only needs in build app) <br>

[»] **Install In Linux**:
1) you need to download `/build` directory first.
2) open terminal in file and run this commands:
```bash
bash ./install.sh 
```
3) application installed on your system
4) you need *sudo for install application <br>

[»] **Install In Windows**

1) you need to download `/build` directory first.
2) open `install.bat` file with adminstrator perms.
3) application installed on your system.
4) if you open 1.0 version installer in powershel you cant see coUlors. (this bug fixes in next version)

[»] **UnInstall In Linux**
1) you can run this commands on console:
```bash
bash ./uninstall.sh
```
2) zima application uninstalled on your system.

[»] **UnInstall In Windows**
1) you can run ``uninstall.bat`` file on your system
2) run file as adminstreator perm.
3) please dont open this file on powershell

[»] **Useage Application**
- you can run commands in console with ``zc`` prefix.
- for open help menu: ``zc --help``.
- you can see your app version: ``zc --version``.
- run commands on this app: ``zc ${command}``

[»] **Build Application**
if you need build this application? no problem run this road-map:
1) you need to download [golang](https://go.dev) compiler
2) this app maded in 1.26.0 version. (can use 1.25)
3) you need download all of project files.
4) run this command on terminal:
```bash
go build -o zc .
```
5) copy compiled file and go ro ``/build`` directory
6) delete old zc or zc.exe file
7) replace new zc of zc.exe file
8) follow install road map on file header
